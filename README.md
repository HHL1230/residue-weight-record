# 殘餘重量記錄自動化系統 (Residue Weight Record)

本專案提供一套基於 Excel VBA 的自動化解決方案，用於管理與紀錄殘餘重量資料。主要功能包含多使用者即時同步、自動備份至 NAS 以及資料覆蓋保護機制。

## 主要功能

### 1. 背景即時同步 (`BackgroundSync`)
- **功能描述**：針對 Excel 的「共同作業 (Multi-User Editing)」模式進行優化。
- **運作機制**：系統啟動後會開啟一個背景計時器，每 10 秒自動執行一次儲存 (`Save`) 動作，確保不同使用者編輯的資料能及時同步至主檔案，解決多人在同一台電腦或不同端點開啟時的資料落差問題。

### 2. 資料覆蓋保護 (`DataProtection`)
- **功能描述**：防止使用者在錄入資料時無意中覆寫已存在的數據。
- **運作機制**：透過攔截 `Worksheet_Change` 事件，當偵測到目標儲存格（如 `C3:E34`, `M3:O34`）已有資料且被修改時，會彈出對話框詢問「確認修改資料？」。若使用者選擇「否」，則系統會利用 `Undo` 功能還原回原始數值。

### 3. 自動備份系統 (`AutoBackup`)
- **功能描述**：確保資料安全性，定期並於關閉檔案時進行遠端備份。
- **遠端路徑**：`\\10.213.74.205\InstrRawData\FCM\backup\`
- **運作機制**：
    - 在檔案關閉前，系統會自動將檔案另存新檔至指定 NAS 路徑。
    - 備份檔名會自動附加時間戳記（例如：`主檔名_20260409_173022.xlsm`），以便追蹤歷史紀錄。
    - 具備錯誤處理機制，若 NAS 連線中斷會跳出警示，確保使用者知曉備份狀態。

## 安裝與使用

1. 開啟目標 Excel 活頁簿（需存為 `.xlsm` 格式）。
2. 按下 `Alt + F11` 進入 VBA 編輯器。
3. 根據檔案名稱將相關程式碼匯入/貼至對應位置：
    - `AutoBackup(ThisWorkbook).vb` -> 貼至 **ThisWorkbook** 物件。
    - `DataProtection(Sheet).vb` -> 貼至對應的 **工作表 (Sheet)** 物件。
    - `BackgroundSync(Module).vb` -> 建立一個新 **模組 (Module)** 並貼入。
4. 儲存並重新開啟檔案即可啟動自動化功能。

## 注意事項
- 檔案必須儲存在啟用巨集的活頁簿格式 (`.xlsm`)。
- 備份功能依賴與 NAS (`10.213.74.205`) 的網路連線。
- 同步功能僅在檔案開啟且非最小化狀態下積極運作。
